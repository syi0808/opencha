Bundled fonts for OpenCHA challenge rendering.

- NotoSans-Bold.ttf: Noto Sans Bold, SIL Open Font License 1.1
- NotoSerif-Bold.ttf: Noto Serif Bold, SIL Open Font License 1.1
- Anton-Regular.ttf: Anton, SIL Open Font License 1.1
- Oswald-Bold.ttf: Oswald Bold, SIL Open Font License 1.1

The challenge renderer parses these TTF files locally and rasterizes the real
glyph outlines into ASCII density cells. No network font loading is used.
